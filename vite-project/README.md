Personal Project Showcase App

A modern, responsive React application for showcasing creative projects. Features a clean card-based layout, dynamic project addition, and real-time search filtering.

Features:
Add Projects: Dynamically add new projects with title & description
Live Search bar projects instantly as you type
Responsive Design: Adapts seamlessly from mobile to desktop
Modern UI Smooth hover effects, card grid layout, and clean typography

Technologies:
React
Vite
CSS(Custom styling)

Usage:
Use the form to add new projects
Type in the search bar to filter the list
View projects in a responsive grid layout
